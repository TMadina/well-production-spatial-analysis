# Well Production Spatial Analysis

This project demonstrates spatial and statistical analysis of synthetic well data.

## Data
- `wells.csv` — synthetic dataset of 186 wells with depth and production
- QGIS project contains maps:
  - Depth gradient map
  - Production heatmap
  - K-means clusters

## Python Analysis
- `pygis_script.py` — PyQGIS automation
- `colab_analysis.ipynb` — scatter plot of depth vs production with trendline and correlation coefficient

## Analysis Highlights
- Visualized spatial distribution of well depth and production
- Identified high- and low-production clusters
- Found positive correlation between depth and production
